# 2. 모터 권선 통합 LCL IOBC dq 임피던스·안정 해석 기술

## 헤드메시지
- EV 구동모터 권선을 LCL 필터에 통합한 3상 IOBC의 입력 임피던스 특성과 제어 안정 한계를 정량 규명
- dq 소신호 모델·능동 댐핑·전류/전압 루프 포함 임피던스 모델과 일반화 나이퀴스트로 그리드 임피던스 변화 하 안정 조건과 고조파 억제 성능 비교 검증

## 기술목적
- 모터 권선 통합 LCL 기반 VSC의 입력 임피던스 거동과 제어 루프 안정 조건을 체계적으로 규명
- 그리드 임피던스 변화와 전압 피드포워드 적용 시 발생 가능한 불안정 요인을 식별

## 기존 기술의 문제점
- 통합 LCL은 모터 파라미터로 그리드측 인덕턴스가 고정되어 공진 주파수와 제어 여유가 예측 곤란
- 그리드 임피던스 변화 시 전류 루프에서 음의 임피던스 영역이 확대되어 진동과 불안정 위험

## 제안 기술의 구체적인 방법 혹은 기술컨셉
1) 모터 권선 통합 LCL 구조와 dq 소신호 모델링
   - 부스트 정류기 평균모델을 dq 변환해 제어-전류 전달 특성의 소신호 모델 도출
   - 모터 권선을 그리드측 인덕터로 등가화해 그리드측 인덕터와 등가저항을 포함한 LCL 어드미턴스 행렬 구성
   - 컨버터 지연을 2차 패드 근사로 모델링해 제어 지연의 위상 여유 영향 반영

2) 능동 댐핑·전류·전압 제어 루프 합성
   - 그리드전류 피드백 고역통과 보상으로 LCL 공진 모드 감쇠
   - PI 전류 제어기와 샘플링·연산 지연을 포함해 대역폭과 위상여유를 설계
   - DC 전압 PI와 전류 기준 생성으로 역률1 운전 및 외부 루프 결합 구현

3) 입력 임피던스 도출과 안정성 판별 절차
   - 제어 미적용, 전류루프 폐루프, 전압루프 폐루프 순으로 입력 어드미턴스를 단계별 계산
   - PCC 전압 피드포워드 포함 시 임피던스 변화와 위상 교차 특성 비교
   - 그리드 임피던스와 입력 임피던스 교차점의 위상 조건으로 잠재 불안정 구간 식별

4) GNC 기반 임피던스 민감도 분석과 HIL 검증
   - 임피던스 비율의 일반화 나이퀴스트 기준으로 폐루프 극점 위치와 안정 여부 판별
   - 그리드 인덕턴스 1μH~3mH 범위에서 전류루프 대비 전압루프 폐루프의 안정성 평가
   - Typhoon HIL 실시간 시험으로 THD, 역률, 부하 스텝 및 임피던스 스텝 응답 검증

## 개선효과
- 통합 LCL은 공진 주파수가 약2.22kHz에서1.65kHz로 이동하고 중고주파 임피던스가 높아 스위칭 고조파 억제 우수
- HIL에서 그리드 인덕턴스 0→3mH 변화에도 DC 전압 유지와 전류 파형 안정 확인, 과도 시 진동은 관찰
- 정격 운전에서 그리드 전류 THD 1.43%와 역률1 달성
